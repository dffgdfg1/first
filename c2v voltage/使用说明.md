# 电压电流对比图工具

## 功能概述

本工具是一个带图形界面的Python脚本，支持多段数据上传（可添加日期标签），自动处理数据后生成电压电流曲线对比HTML图表。

## 核心功能

### 1. 图形用户界面
- **文件上传区**：按钮触发文件选择，支持.csv/.txt等文本格式
- **日期选择器**：点击"📅 选择日期"按钮可快速选择日期，无需手动输入
- **已上传列表**：显示已选文件路径+对应日期标签，支持删除
- **历史记录**：查看之前生成的所有图表，一键打开
- **处理按钮**："开始处理"触发数据解析和图表生成
- **状态提示**：显示上传进度、错误信息

### 2. 数据预处理
- **时间列格式化**：
  - 支持Excel序列值转换（基准日期1899-12-30）
  - 支持HH:MM:SS、HH:MM:SS.mmm格式
  - 支持YYYY-MM-DD HH:MM:SS格式
  - 自动将时间戳与用户输入的日期组合

### 3. 多段数据整合与对比
- **电压曲线**：使用深红色显示（左Y轴）
- **电流曲线**：使用深蓝色显示（右Y轴）
- 不同日期数据用不同线型区分
- 时间轴统一，所有数据在同一时间轴上对齐

### 4. HTML图表核心功能（基于Plotly）
- **双Y轴曲线**：左Y轴显示电压(V)，右Y轴显示电流(A)
- **颜色区分**：电压用深红色，电流用深蓝色，一目了然
- **图例简化**：只显示"电压"和"电流"，不显示多余信息
- **鼠标悬停**：显示完整时间戳、电压值、电流值
- **图例切换**：点击图例可隐藏/显示对应曲线
- **X轴交互**：
  - 鼠标拖拽框选放大
  - 滚轮缩放
  - 双击重置视图
- **时间范围显示**：X轴两端显示完整的开始和结束日期时间
- **日期联动**：滑动滑块时，X轴两端日期实时更新
- **图片保存**：集成"下载为PNG"按钮，支持高清导出（144dpi）
- **Y轴从0开始**：图表更直观，避免误解

### 5. 历史记录功能
- **自动记录**：每次生成的图表自动保存到历史记录
- **快速查看**：双击历史记录即可打开对应的HTML图表
- **文件夹管理**：���键打开图表所在文件夹
- **历史清理**：支持清空历史记录（不会删除已生成的HTML文件）

## 使用方法

### 方法一：双击批处理文件（推荐）
1. 双击 `启动工具.bat`
2. 等待应用启动

### 方法二：命令行运行
```bash
python voltage_current_plotter.py
```

## 操作步骤

1. **选择文件**
   - 点击"📁 选择文件 (CSV/TXT)"按钮
   - 选择一个或多个CSV/TXT文件
   - 界面会显示"已选择 X 个文件"

2. **选择日期（新功能）**
   - 点击"📅 选择日期"按钮
   - 在弹出的对话框中选择年、月、日
   - 点击"确定"完成日期选择

3. **添加到列表**
   - 点击"➕ 添加到列表"按钮
   - 文件会显示在左侧的"已上传文件列表"中

4. **重复添加**
   - 如果有不同日期的数据，重复步骤1-3
   - 每次选择文件后选择对应的日期

5. **开始处理**
   - 点击"开始处理并生成图表"按钮
   - 等待处理完成，浏览器会自动打开生成的HTML图表

6. **查看历史记录**
   - 右侧"历史记录"区域显示所有生成的图表
   - 双击某条记录可打开对应的HTML文件
   - 点击"📂 打开文件夹"可打开文件所在目录

## 数据格式要求

CSV文件需包含以下列（列名可灵活匹配）：
- **时间列**：列名包含"时间"或"time"
  - 支持格式：HH:MM:SS、HH:MM:SS.mmm、YYYY-MM-DD HH:MM:SS、Excel序列值
- **电压列**：列名包含"电压"或"voltage"
- **电流列**：列名包含"电流"或"current"

示例：
```csv
时间,循环次数,电压(11.76-12.24),电流(70.00-130.00)
17:29:29.565,1,12.06,92.29
17:29:30.575,1,12.06,92.04
17:29:31.569,1,12.06,92.29
```

## 输出文件

生成的HTML文件命名格式：
```
电压电流对比图_YYYYMMDD_HHMMSS.html
```

例如：
```
电压电流对比图_20240120_153045.html
```

历史记录保存在：`chart_history.txt`

## 常见问题

**Q: 提示"未找到时间列"？**
A: 请确保CSV文件中有包含"时间"或"time"关键词的列名。

**Q: 时间显示不正确？**
A: 请检查"数据日期"是否正确选择。

**Q: 浏览器没有自动打开？**
A: 生成的HTML文件保存在脚本同一目录下，可以手动用浏览器打开，或在历史记录中双击打开。

**Q: 如何隐藏某条曲线？**
A: 点击图例中对应的曲线名称即可隐藏/显示。

**Q: 电压和电流为什么用不同颜色？**
A: 这样可以更清晰地区分两种物理量，深红色代表电压，深蓝色代表电流。

**Q: 如何查看之前生成的图表？**
A: 在右侧"历史记录"区域双击对应的记录即可打开。

**Q: 历史记录太多怎么办？**
A: 点击"🗑️ 清空历史"可以清空历史列表（不会删除已生成的HTML文件）。
